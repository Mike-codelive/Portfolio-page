Real State

full-stack application involves implementing Server-side Rendering (SSR) approach using API Routes provided by Next.js. Facilitating the deployment of serverless functions and management of backend logic without the necessity of setting up a separate server. This capability empowers us to perform CRUD operations with a high degree of flexibility on MongoDB, a database system designed to scale horizontally. Its JSON-like document structure aligns well with the object-oriented programming paradigm. On the front end, React is utilized to efficiently manage the application through modular components. Additionally, Tailwind CSS is employed for styling, ensuring a streamlined and cohesive visual presentation.

Technologies: React, Next.js, JavaScript, Tailwind, MongoDB
